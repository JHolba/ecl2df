wcon¶

This module extracts information from WCONHIST, WCONINJE, WCONINJH and WCONPROD from an Eclipse deck.

from ecl2df import wcon, EclFiles

eclfiles = EclFiles("MYDATADECK.DATA")
dframe = wcon.df(eclfiles)
